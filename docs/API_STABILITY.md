# API Stability

loxguard is currently `v0.x` and the API is not yet stable.

What to expect in `v0.x`:
- breaking changes may occur between minor versions
- types, enums, and function signatures may change as the library evolves
- CSV/report formats may evolve; treat them as diagnostic outputs rather than long-term stable storage formats

For current verified behavior, see `docs/EVIDENCE_MATRIX.md`.
