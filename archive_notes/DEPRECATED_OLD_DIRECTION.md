# Deprecated old directions

Older pack versions focused on:

- MPU-only runtime,
- generic resilience framework,
- pure integration stack.

These are not the current primary direction.

Current direction:

```text
Guard Blocks first.
Checked Guard Blocks as the first strong differentiator.
Failure pipeline underneath.
Existing libraries as accelerators.
Instrumentation/RTOS/MPU as optional future backends.
```
